# DataPrepUtils
Este proyecto implementa un paquete de Python para la preparación básica de datos.

## Funcionalidades
Incluye funciones para:

- limpieza de datos
- normalización
- cálculos estadísticos

El paquete está organizado en módulos:

cleaning.py
normalization.py
statistics.py

### Cleaning
Funciones para limpieza de datos:
- eliminación de valores nulos
- reemplazo de valores faltantes

### Normalization
Funciones de transformación de datos:
- normalización Min-Max
- estandarización Z-score

### Statistics
Funciones estadísticas:
- cálculo de moda
- media armónica
- media geométrica

Antes de ejecutar el proyecto, instala las librerías necesarias usando el archivo requirements.txt.

``` pip install -r requirements.txt```

Esto instalará las dependencias necesarias como:

- pandas
- numpy

## Uso
El archivo main.py demuestra el uso del paquete.

Para ejecutar el ejemplo incluido en el proyecto:

``` python main.py```
